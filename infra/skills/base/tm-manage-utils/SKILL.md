# tm-manage-utils — 유틸 스킬 관리 (리다이렉트)

유틸 스킬 관리는 이제 `tm-customize`(스킬 영역)로 통합됐습니다 → **`tm-customize`** 를 사용하세요.

`tm-customize` 진입 후 "스킬" 영역을 선택하면 util add/remove/list 동사를 그대로 사용할 수 있습니다.
