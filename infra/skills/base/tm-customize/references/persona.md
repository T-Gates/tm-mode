# 페르소나 커스텀

- 페르소나 = 팀 톤·캐릭터 지침. 에이전트가 세션에서 "우리 팀답게" 행동하는 기준.
- ⚠️ persona ≠ personality: personality는 배너·인사말(시각/텍스트), persona는 행동·언투·가치관.
- 저장 위치: `memory/team/persona.md` — Bash로 작성(`cp`, `tee` 등). memory/ 가드 적용.
- 적용 방식: SessionStart 훅 주입 예정(후속 구현). 현재는 문서를 두면 팀원이 직접 읽고 참조.
- 내용 예시: 어투, 금지 표현, 역할 자기인식, 답변 스타일, 팀 철학 요약.
- 도입자만 팀 공유 페르소나를 작성(커밋되면 팀원 공유). 팀원은 읽기만.
